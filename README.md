# Payroll-Management-System

## Overview
A console-based payroll system developed in C++.  
Allows adding employees, viewing details, calculating payroll, and saving records to a file.

## Features
- Add employee records (ID, name, surname, salary, benefits, deductions)  
- View employee details by ID  
- Calculate total payroll  
- Save records to file  
- Menu-driven interface

## Concepts Application
- Classes & structs  
- File handling  
- Loops, conditionals, functions  
- Object-oriented design

## Run
1. Compile using a C++ compiler  
2. Run the executable  
3. Follow the menu instructions
